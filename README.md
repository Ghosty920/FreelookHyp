## Freelook
Allows you to freely look around.

![Minimal Compact Powered by OneConfig](https://polyfrost.org/media/branding/badges/badge_4.svg)

---

### Features
- First person & Third person.
- Snaplook mode. (Auto-enabled on Hypixel)
- Unlockable pitch.
- Key bind.

### Planned features
- Cinematic Camera.
- Camera animations.
- Custom FOV.
